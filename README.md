
# Explore US Bikeshare Data project 
In this project, I used Python to interactivlely explore bikeshare data for three major cities in the US.  
## Files used
bikeshare.py chicago.csv new_york_city.csv washington.csv
## Information about how to use your project
-You should have Python 3, NumPy, and pandas installed using Anaconda
-A text editor, like Sublime(opens in a new tab) or Visual Studio Code(opens in a new tab)
-A terminal application (Terminal on Mac and Linux or Git Bash on Windows).
